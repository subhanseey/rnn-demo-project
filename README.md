# IMDB Movie Review Sentiment Analysis
A Deep Learning web application that classifies movie reviews as **Positive** or **Negative** using a Simple RNN trained on the IMDB dataset.

#Tech Stack
* Python
* TensorFlow / Keras
* Streamlit
* NumPy

# Live Demo
https://rnn-demo-project.onrender.com

# Features
* Real-time sentiment prediction
* Simple and interactive Streamlit UI
* Deep Learning-based text classification
